DOM

¿Que es el DOM?

El DOM, se refiere al modelo de objetos de un documento. Es una representación jerárquica de la estructura de un documento HTML, que proporciona una interfaz programática para interactuar y manipular el contenido de la página web mediante scripts, como JavaScript.

Obtener un elemento o elementos en JavaScript.

Podemos hacerlo con los siguientes métodos:

  • getElementById:
    • - Este método busca y devuelve un elemento del documento HTML según su atributo "id". Dado que el "id" de un elemento debe ser único en la página, suele utilizarse para acceder a un elemento específico.

    let elemento = document.getElementById("idDelElemento");

  • querySelector:
    • - Este método devuelve el primer elemento que coincide con un selector CSS específico. Puedes utilizar cualquier selector CSS válido.

    let elemento = document.querySelector("selectorCSS");

  • querySelectorAll:
    • - Similar a querySelector, pero en lugar de devolver solo el primer elemento, devuelve todos los elementos que coinciden con el selector.

    let elementos = document.querySelectorAll("selectorCSS");

NODOS

Son objetos que representan partes de un documento HTML. El DOM organiza la estructura del documento en una jerarquía de nodos, donde cada elemento, atributo, y fragmento de texto en el documento es un nodo.

  • Nodo texto
    • - Representa el contenido textual de un elemento. Por ejemplo, si tienes un párrafo "Texto aquí", el "Texto aquí" sería un nodo de texto.
  • Nodo elemento
    • - Representa un elemento HTML y puede tener nodos secundarios (hijos), como nodos de texto o atributos.
  • Nodo atributo
    • - Representa un atributo de un elemento. Cada atributo del elemento es considerado un nodo separado.

TEXTCONTENT

Es una propiedad en JavaScript que permite acceder al contenido de texto de un elemento y también modificarlo. Esta propiedad devuelve o establece el contenido textual de un nodo y todos sus descendientes.

DOMTRAVERSING

Es el proceso de moverse a través de la estructura del Document Object Model (DOM) para acceder, modificar o manipular elementos HTML. Los métodos más comunes son:

  • parentElement
  • nextSibling
  • childElement

CREATEELEMENT

Esta función se utiliza para crear un nuevo elemento HTML en el DOM.

INNERHTML

Esta propiedad es una forma de acceder y modificar el contenido HTML de un elemento. Al usar innerHTML, puedes obtener el HTML interno de un elemento o establecer un nuevo contenido HTML.
